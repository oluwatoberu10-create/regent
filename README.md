# REGENT — Business Automation Agency Website

A single-page, animated marketing site for **REGENT**, a done-for-you business automation agency.
It showcases the outcomes of automation (handling enquiries, booking appointments, following up
on leads, invoicing) with pricing tiers, testimonials, and a Calendly booking integration.

## Features
- Regal old-gold luxury aesthetic with vibrant orange + cyan **neon** accents
- Looping background video + animated node-network canvas hero
- Live automation pipeline visualization
- Scroll progress bar, custom cursor glow, magnetic buttons, 3D tilt cards, parallax
- Pricing tiers (Spark / Engine / Sovereign) with monthly⇄annual toggle
- Full automation catalogue: Social Media, Lead Generation, CRM, Voicebot, Chatbot, Invoicing
- Testimonial marquee + scroll-triggered count-up metrics
- Right-edge **Book a call** tab → Calendly popup
- Contact form that opens an email to the agency

## Setup
Open `index.html` in any browser. Before going live, edit the config at the bottom of `index.html`:

```js
const CALENDLY_URL  = 'https://calendly.com/oluwatoberu10/30min'; // your real Calendly link
const CONTACT_EMAIL = 'oluwatoberu10@gmail.com';
```

## Deploy (GitHub Pages)
Settings → Pages → Branch: `main` / root. The site will be served at
`https://oluwatoberu10-create.github.io/regent/`.
